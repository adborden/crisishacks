About Django Hacks

Django Hacks solves the problem in the Django community of being able to easily identify good apps, frameworks, and hacks. Ever want to know which is the most popular or well supported Django blog, content management system, or api tool? Django Hacks solves that problem for you!

What is a Django hack?

A Django hack is anything that is involved in the Django ecosphere that can be stored on a repository such as Github or Bitbucket. If it can be stored in PyPI, that's even better!

When did you code this?

Django Hacks was conceived for and coding during Django Dash 2010. Since then we've been feverishly adding more features, improving the design, and trying to make data entry as easy/intuitive as possible.

Who built this thing?

Daniel Greenfeld and Audrey Roy are the creators of this project. Additional contributing members are detailed in the team page.

Can I help?

Django Hacks is an volunteer based open source effort. Please submit any ideas or issues and also fork and submit back patches and enhancements.
