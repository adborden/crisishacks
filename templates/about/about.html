About Crisis Hacks

There are a lot of passionate developers who create open source software for the crisis response communities, often at hackathons like Random Hacks of Kindness. However there is not a good way to track what hacks are being worked on, and by whom. Crisis Hacks is a simple website designed to solve this problem.

Who Built This?

This site is largely based on the excellent Django Packages site which was created by Daniel Greenfeld and Audrey Roy. The codebase was customized slightly by Josh Livni, Ariel Núñez, and others for use with the crisis hacking community.

What is a Crisis Hack?

A crisis hack is any open source project, script, or other software that aims to solve issues facing the crisis response community. To be eligible for inclusion in this site, the software must be publicly available on a site like Google Code, GitHub or other similar repositories.

Can I help?

Django Hacks is an volunteer based open source effort. Please submit any ideas or issues and also fork and submit back patches and enhancements.
